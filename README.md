# openclaw-react-repo

One-page React app that shows **current weather for the user's region**.

## How it works
- Reads **browser settings**: locale + time zone
- Requests **Geolocation** permission to get latitude/longitude
- Fetches current weather from **Open-Meteo**

## Dev
```bash
npm i
npm run dev
```

## Deploy
Deployed to GitHub Pages via GitHub Actions.
